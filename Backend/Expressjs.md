### 👨‍🏫 Express.js 
> Infraestructura web rápida, minimalista y flexible para Node.js

### Instalación

Para instalar la librería, se hace uso de [*`npm`*](https://www.npmjs.com/) *(node package manager)*
```bash
$ npm install express --save
```
